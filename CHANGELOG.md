# Changelog

All notable changes to the Mr Gas website will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-02-04

### Added
- Initial release of Mr Gas website
- Responsive mobile-first design
- Hero section with trust badges
- About section with company values
- Services section showcasing 6 main offerings
- Why Choose Us section with 6 key differentiators
- Customer reviews section with 6 testimonials
- Contact section with form and business information
- Professional navigation with mobile menu
- Smooth scroll functionality
- Form validation and submission handling
- SEO optimization (meta tags, semantic HTML)
- Accessibility features (ARIA labels, keyboard navigation)
- Performance optimizations (minimal dependencies, lazy loading ready)
- Cross-browser compatibility
- Print-friendly styles
- Comprehensive documentation:
  - README.md
  - DEPLOYMENT.md
  - CUSTOMIZATION.md
  - LICENSE
  - robots.txt
  - sitemap.xml
  - .gitignore

### Design
- Modern, clean layout with strong visual hierarchy
- Brand colors: Red (#DC0000), Black (#000000), White (#FFFFFF)
- Inter font family for professional typography
- Card-based design for services and features
- Subtle animations and transitions
- Industrial, bold, trustworthy aesthetic

### Technical
- Pure HTML5, CSS3, and Vanilla JavaScript
- No external dependencies (except Google Fonts)
- CSS Grid and Flexbox for layouts
- CSS Custom Properties for easy theming
- Intersection Observer for scroll animations
- Mobile-first responsive breakpoints

## How to Update This File

When making changes to the website:

1. Add entry under "Unreleased" section
2. When deploying, move changes to new version section
3. Update date and version number

### Categories
- **Added** - New features
- **Changed** - Changes to existing functionality  
- **Deprecated** - Soon-to-be removed features
- **Removed** - Removed features
- **Fixed** - Bug fixes
- **Security** - Security improvements
